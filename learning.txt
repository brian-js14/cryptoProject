Creando componentes:
un componente solo devuelve una cosa, si queremos devolver más de una cosa se debe importar un view
Toda aplicación y componentes debe llevar:
    import React from "react";
    import { TextInput, Button, Text, View } from "react-native"; // Este es un ejemplo de como puede ir

-Los componentes se declaran de la siguiente manera:
export default function Saludar(props) {} --> importante que el nombre de la función, la primera letra debe ser en mayúsculas
para poder definir propiedades por defecto se debe enviar nombreFuncion.defaultProps {
    atributos, ej:
    nombre: "",
    apellido: ""
}

PropTypes => se definen métodos obligatorios
Saludar.propTypes = {
  firstname: PropTypes.string.isRequired,
  lastname: PropTypes.string,
};
